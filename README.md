# Stats Skills — Claude AI Protocollen

Herbruikbare analyse-skills voor **Statistiek IV — Multivariate Data-Analyse** (VUB, Theuns et al., 2016).  
Elke skill is een gestructureerd prompt-protocol met R- en SPSS-code, diagnostische beslisbomen en APA-rapportagesjablonen.

## Gebruik

Elke skill is ontworpen als **context-injectie voor Claude**.  
Kopieer de inhoud van een skill-bestand in je gesprek met Claude als systeeminstructie of eerste prompt.
